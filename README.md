# Personal Portfolio Website

This is my first complete web project built from scratch as an IT student.
The project represents my personal portfolio and showcases my skills, projects, and basic information in a clean and organized way.

##  Features
- Responsive multi-page website
- Projects showcase
- Team / members section
- Contact page (under development)
- Clean file structure and separation of concerns

##  Tech Stack
- HTML5
- CSS3
- JavaScript (Vanilla JS)
- Node.js (for future backend expansion)
